# Проект "Личные заметки"

## Описание

Notes — это веб-приложение для создания, хранения заметок. Для просмотра и создаия заметок необходимо зарегистрироваться. Каждый пользователь может видеть только свои заметки.

## Функциональности

- Авторизация
- Хранение заметок

## Технологии

- React.js
- React Query
